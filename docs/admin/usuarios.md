# Usuarios

Administra los usuarios que acceden al sistema.

## Acceder al Módulo
Menú lateral → **Administración** → **Usuarios**

## Funciones Disponibles

### Ver Lista
Visualiza todos los usuarios con su partner, roles y estado.

### Crear Usuario
1. Clic en **Nuevo Usuario**
2. Completa: Nombre, Email, Contraseña, Partner, Roles
3. Guarda

### Editar Usuario
1. Clic en **Editar**
2. Modifica los campos (deja contraseña vacía para mantener la actual)
3. Actualiza

### Activar/Desactivar
Edita el usuario y cambia el campo Estado.

## Roles que puedes asignar
- Asociado Administrador
- Asociado Vendedor
- user

> **Nota**: No puedes asignar rol de super admin o admin.
