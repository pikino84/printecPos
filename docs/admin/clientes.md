# Clientes

Gestiona los clientes finales para cotizaciones.

## Acceder al Módulo
Menú lateral → **Administración** → **Clientes**

## Funciones

### Crear Cliente
1. Clic en **Nuevo Cliente**
2. Completa: Nombre, Email, Teléfono, RFC, Dirección
3. Guarda

### Editar/Eliminar
Usa los botones de acción junto a cada cliente.

### Ver Detalle
Clic en **Ver** para ver información completa y cotizaciones asociadas.

## Tips
- El email es importante para enviar cotizaciones
- Cada partner ve solo sus propios clientes
- Como Admin, puedes ver clientes de todos los partners
