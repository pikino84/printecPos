# Cotizaciones

Crea y gestiona cotizaciones para clientes.

## Acceder al Módulo
Menú lateral → **Productos** → **Mis Cotizaciones**

## Crear Cotización

1. Agrega productos al carrito desde el **Catálogo**
2. Ve al **Carrito**
3. Selecciona Cliente y Razón Social
4. Clic en **Generar Cotización**

## Acciones

### Ver
Visualiza el detalle completo de la cotización.

### Descargar PDF
Genera el documento PDF para compartir.

### Enviar por Correo
Envía automáticamente al email del cliente.

### Editar/Clonar
Carga los productos al carrito para modificar o crear una nueva.
