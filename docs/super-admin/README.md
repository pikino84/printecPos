- [Permisos](./permisos.md) - Gestión de permisos del sistema

- [Clientes](./clientes.md) - Gestión de clientes finales

## Accesos Rápidos

| Acción | Ruta |
|--------|------|
| Ver todos los partners | `/partners` |
| Crear nuevo usuario | `/users/create` |
| Gestionar roles | `/roles` |
| Ver actividad | `/activity-logs` |
| Configurar pricing | `/pricing-settings` |

## Tips Importantes

1. **Antes de crear usuarios**, asegúrate de tener los roles correctamente configurados
2. **Revisa el historial de actividad** regularmente para detectar anomalías
3. **Los cambios en pricing-settings** afectan a todo el sistema, procede con precaución
4. **Al desactivar un partner**, sus usuarios también perderán acceso
