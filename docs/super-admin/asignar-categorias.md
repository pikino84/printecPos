# Asignar Categorías

Este módulo permite mapear las categorías de los proveedores a las categorías internas de Printec.

## Acceder al Módulo

1. En el menú lateral, haz clic en **Productos**
2. Selecciona **Asignar Categorías**

## Vista de Mapeo

La interfaz muestra:
- **Categorías del Proveedor**: Categorías originales de cada proveedor
- **Categoría Printec**: Categoría interna a la que se asigna
- **Productos**: Cantidad de productos en esa categoría

## Asignar una Categoría

1. Busca la categoría del proveedor que deseas mapear
2. En el selector de **Categoría Printec**, elige la categoría destino
3. El cambio se guarda automáticamente

## Casos de Uso

### Unificar Categorías Similares
Si el proveedor A tiene "Bolígrafos" y el proveedor B tiene "Plumas", puedes asignar ambas a una categoría interna "Instrumentos de Escritura".

### Reorganizar Productos
Puedes mover productos de una categoría a otra cambiando el mapeo.

## Tips

- Revisa los mapeos cuando se agregan nuevos proveedores
- Categorías bien mapeadas mejoran la experiencia de búsqueda
- Los productos sin categoría asignada aparecen en "Sin categoría"
- Usa el filtro para encontrar categorías específicas
