## Tu Flujo de Trabajo

### 1. Registrar al Cliente
Si es un cliente nuevo, regístralo primero en el módulo de Clientes.

### 2. Explorar el Catálogo
Busca los productos que necesita el cliente usando filtros y búsqueda.

### 3. Agregar al Carrito
Selecciona productos, cantidades y agrégalos al carrito.

### 4. Generar Cotización
Revisa el carrito, selecciona el cliente y genera la cotización.

### 5. Enviar al Cliente
Envía la cotización por correo o descarga el PDF para compartirlo.

## Los Precios

Los precios que ves en el catálogo ya incluyen:
- El descuento por nivel de pricing de tu organización
- El markup configurado por tu administrador

Estos son los precios finales que verá tu cliente en la cotización.

## Tips

- Mantén actualizados los datos de tus clientes
- Usa los filtros para encontrar productos rápidamente
- Puedes clonar cotizaciones anteriores para ahorrar tiempo
- Contacta a tu administrador si necesitas cambios en usuarios o configuración
