# Documentación de Usuario - PrintecPOS

Bienvenido a la documentación de PrintecPOS. Esta guía te ayudará a entender y utilizar todas las funcionalidades del sistema según tu rol.

## Índice por Rol

Selecciona tu rol para acceder a la documentación correspondiente:

### [Super Admin](./super-admin/README.md)
Acceso completo a todas las funcionalidades del sistema, incluyendo:
- Gestión de Partners
- Gestión de Usuarios, Roles y Permisos
- Configuración del sistema de Pricing
- Historial de Actividad
- Todas las funciones de administración

### [Admin](./admin/README.md)
Gestión administrativa del sistema (sin acceso a roles/permisos/configuración de pricing):
- Gestión de Partners
- Gestión de Usuarios
- Gestión de Clientes
- Catálogo y Cotizaciones
- Distribuidor (Razones Sociales, Cuentas Bancarias)

### [Asociado Administrador](./asociado-administrador/README.md)
Gestión de tu organización como distribuidor:
- Gestión de tu equipo de usuarios
- Clientes y Cotizaciones
- Sistema de Pricing (consulta y reportes)
- Razones Sociales y Cuentas Bancarias
- Configuración de Markup/Ganancia
- Productos Propios

### [Asociado Vendedor](./asociado-vendedor/README.md)
Funciones de venta y atención a clientes:
- Gestión de Clientes
- Catálogo de Productos
- Creación de Cotizaciones
- Consulta de Productos Propios

---

## Primeros Pasos

1. **Inicia sesión** con tu correo y contraseña proporcionados por el administrador
2. **Completa el tutorial interactivo** que aparece automáticamente la primera vez
3. Si necesitas ver el tutorial nuevamente, haz clic en **"Ver Tutorial"** en el menú de usuario (esquina superior derecha)

## Soporte

Si tienes dudas o problemas técnicos, contacta al administrador del sistema.
