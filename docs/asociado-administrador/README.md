# Guía del Asociado Administrador

Como Asociado Administrador, gestionas tu organización distribuidora: equipo de usuarios, clientes, cotizaciones y tienes acceso al sistema de pricing.

## Módulos Disponibles

### Administración
- [Mis Usuarios](./mis-usuarios.md) - Gestión de tu equipo
- [Clientes](./clientes.md) - Tu cartera de clientes
- [Mis Almacenes](./mis-almacenes.md) - Almacenes de tu organización

### Pricing
- [Dashboard de Pricing](./pricing-dashboard.md) - Tu nivel actual
- [Niveles de Precio](./pricing-tiers.md) - Niveles disponibles
- [Mi Pricing](./mi-pricing.md) - Tu historial de pricing

### Distribuidor
- [Razones Sociales](./razones-sociales.md) - Tus entidades fiscales
- [Cuentas Bancarias](./cuentas-bancarias.md) - Cuentas para pagos
- [Mi Ganancia](./mi-ganancia.md) - Configuración de markup

### Productos
- [Mis Categorías](./mis-categorias.md) - Organización personalizada
- [Productos Propios](./productos-propios.md) - Tus productos
- [Catálogo](./catalogo.md) - Catálogo de productos
- [Cotizaciones](./cotizaciones.md) - Gestión de cotizaciones

## Accesos Rápidos

| Acción | Ruta |
|--------|------|
| Ver mi equipo | `/mis-usuarios` |
| Ver clientes | `/clients` |
| Configurar markup | `/mi-ganancia` |
| Ver mi pricing | `/pricing-dashboard` |

## Tu Flujo de Trabajo Típico

1. **Configura tu organización**
   - Agrega razones sociales y cuentas bancarias
   - Configura tu margen de ganancia (markup)

2. **Gestiona tu equipo**
   - Crea usuarios vendedores
   - Asígnales acceso

3. **Atiende clientes**
   - Registra clientes
   - Crea cotizaciones
   - Envía por correo o descarga PDF

4. **Monitorea tu pricing**
   - Revisa tu nivel actual
   - Consulta el progreso hacia el siguiente nivel
