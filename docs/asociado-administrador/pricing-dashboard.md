# Dashboard de Pricing

Visualiza tu nivel de precios actual y el progreso hacia el siguiente nivel.

## Acceder al Módulo
Menú lateral → **Administración** → **Dashboard de Pricing**

## Tu Información

El dashboard muestra:

### Tu Nivel Actual
- Nombre del nivel (ej: Plata, Oro)
- Descuento que recibes sobre precios base

### Progreso
- Volumen de compras acumulado
- Monto para alcanzar el siguiente nivel
- Barra de progreso visual

### Beneficios
- Descuento actual vs siguiente nivel
- Ahorro proyectado al subir de nivel

## Sistema de Pricing

PrintecPOS utiliza un sistema de precios escalonados:

1. **Acumulas compras** a lo largo del tiempo
2. **Alcanzas umbrales** que te suben de nivel
3. **Mejores precios** mientras más compras

### Ejemplo de Niveles

| Nivel | Descuento | Compras Mínimas |
|-------|-----------|-----------------|
| Inicial | 0% | $0 |
| Bronce | 5% | $10,000 |
| Plata | 10% | $50,000 |
| Oro | 15% | $100,000 |

## Tips

- Consulta regularmente tu progreso
- Los precios del catálogo ya reflejan tu descuento
- Tu descuento se aplica automáticamente
