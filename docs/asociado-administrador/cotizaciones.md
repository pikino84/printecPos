# Cotizaciones

Crea, gestiona y envía cotizaciones a tus clientes.

## Acceder al Módulo
Menú lateral → **Productos** → **Mis Cotizaciones**

## Crear una Cotización

### Paso 1: Agregar productos
1. Ve al **Catálogo**
2. Agrega productos al carrito

### Paso 2: Revisar el carrito
1. Clic en el ícono del **Carrito**
2. Ajusta cantidades si es necesario
3. Puedes eliminar productos

### Paso 3: Generar cotización
1. Selecciona el **Cliente** (o crea uno nuevo)
2. Selecciona la **Razón Social** a usar
3. Clic en **Generar Cotización**

## Lista de Cotizaciones

La tabla muestra todas tus cotizaciones con:
- Número y fecha
- Cliente
- Total
- Acciones disponibles

## Acciones

| Acción | Descripción |
|--------|-------------|
| **Ver** | Visualiza el detalle completo |
| **PDF** | Descarga el documento PDF |
| **Enviar** | Envía por correo al cliente |
| **Editar** | Carga al carrito para modificar |
| **Clonar** | Copia para crear nueva cotización |
| **Eliminar** | Elimina la cotización |

## El PDF Incluye

- Logo de tu organización (si está configurado)
- Datos de tu razón social
- Información del cliente
- Detalle de productos y precios
- Totales
- Cuentas bancarias para pago

## Tips

- Revisa los precios antes de enviar
- Mantén actualizados los datos del cliente
- Usa "Clonar" para crear cotizaciones similares rápidamente
- El correo sale desde el servidor configurado en tu razón social
