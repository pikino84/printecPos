# Mi Ganancia (Markup)

Configura el margen de ganancia que se aplicará sobre los precios del catálogo.

## Acceder al Módulo
Menú lateral → **Distribuidor** → **Mi Ganancia**

## ¿Qué es el Markup?

El markup es el porcentaje que agregas sobre el precio base de los productos para obtener tu ganancia.

### Ejemplo
- Precio base del producto: $100
- Tu markup: 15%
- Precio final: $115

## Configurar tu Markup

1. Ingresa el **Porcentaje de ganancia** deseado
2. Clic en **Guardar**

## Cómo Afecta los Precios

- El markup se aplica a todos los productos del catálogo
- Los precios en tus cotizaciones incluyen el markup
- El markup se aplica **después** del descuento de tu nivel de pricing

### Cálculo Completo

```
Precio Base → Descuento por Pricing → + Tu Markup = Precio Final
```

**Ejemplo:**
- Precio base: $100
- Tu nivel de pricing da 10% descuento: $90
- Tu markup de 15%: $90 + 15% = $103.50

## Tips

- Considera tu mercado al definir el markup
- Un markup muy alto puede hacer tus precios poco competitivos
- Revisa periódicamente si tu markup sigue siendo adecuado
- El markup también se aplica a productos propios
